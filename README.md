# School Infrastructure & Enrollment Analytics Dashboard

## Project Overview
This Power BI dashboard analyzes school infrastructure, student enrollment, teacher availability, and educational distribution across different regions.

## Objectives
- Analyze school distribution by state and district
- Evaluate student enrollment trends
- Assess teacher availability and teacher-student ratio
- Compare infrastructure facilities across schools
- Generate actionable insights for educational planning

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- GitHub

## Dashboard Pages
1. Executive Overview
2. School Analytics
3. Student Enrollment Analytics
4. Teacher Analytics
5. Infrastructure Analytics

## Key KPIs
- Total Schools
- Total Students
- Total Teachers
- Total Classrooms
- Teacher-Student Ratio
- Infrastructure Score

## Features
- Interactive Slicers
- Drill Down
- Drill Through
- Dynamic Titles
- Custom Tooltips
- Maps and Treemaps

## Business Insights
- Identified districts with higher school concentration
- Evaluated student enrollment patterns
- Assessed teacher allocation efficiency
- Highlighted infrastructure gaps across regions
